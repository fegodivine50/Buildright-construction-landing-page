BuildRight Construction Landing Page

A modern, responsive landing page for BuildRight Construction, showcasing services, projects, testimonials, and contact information. Built with HTML, CSS, and JavaScript, this project demonstrates responsive design, interactive elements, and clean layout practices.
_____________________________
Features

Fully responsive design for desktop, tablet, and mobile screens

Hero section with call-to-action button

Services section with interactive cards

Projects section highlighting real projects

About section with company overview and features

Testimonials section displaying client feedback

Contact section with a form and company info

Footer with links and social media icons

Hamburger menu for mobile navigation

HTML5

CSS3

JavaScript (for hamburger menu toggle)
